# Q1 (4 điểm): Reflex Agent

Hãy cải thiện `ReflexAgent` trong `multiAgents.py` để agent chơi ổn hơn. Mã reflex agent được cung cấp sẵn có các ví dụ hữu ích về cách dùng các phương thức truy vấn thông tin từ `GameState`. Một reflex agent tốt cần cân nhắc cả vị trí thức ăn và vị trí ghost để chơi hiệu quả. Agent của bạn nên có thể vượt layout `testClassic` một cách dễ dàng và ổn định:
```bash
python pacman.py -p ReflexAgent -l testClassic
```
Hãy thử reflex agent của bạn trên layout mặc định `mediumClassic` với một hoặc hai ghost (tắt animation để chạy nhanh hơn):
```bash
python pacman.py --frameTime 0 -p ReflexAgent -k 1
python pacman.py --frameTime 0 -p ReflexAgent -k 2
```
Agent của bạn chơi ra sao? Nhiều khả năng sẽ thường xuyên chết với 2 ghost trên bản đồ mặc định, trừ khi hàm đánh giá của bạn đủ tốt.

* **Lưu ý:** Nhớ rằng `newFood` có hàm `asList()`.
* **Lưu ý:** Khi làm feature, hãy thử dùng nghịch đảo của các giá trị quan trọng (ví dụ khoảng cách đến thức ăn) thay vì dùng trực tiếp giá trị đó.
* **Lưu ý:** Hàm đánh giá bạn đang viết ở đây là cho cặp state-action; ở các phần sau của project, bạn sẽ đánh giá state.
* **Lưu ý:** Bạn có thể thấy hữu ích khi xem nội dung bên trong của các object để debug. Có thể in biểu diễn chuỗi của object ra màn hình. Ví dụ, bạn có thể in `newGhostStates` bằng `print(newGhostStates)`.

**Tùy chọn:** Ghost mặc định đi ngẫu nhiên; bạn cũng có thể thử loại ghost thông minh hơn một chút bằng `-g DirectionalGhost`. Nếu tính ngẫu nhiên khiến bạn khó đánh giá việc agent có cải thiện hay không, hãy dùng `-f` để chạy với random seed cố định (các lựa chọn ngẫu nhiên giống nhau mỗi game). Bạn cũng có thể chạy nhiều game liên tiếp bằng `-n`. Tắt đồ họa bằng `-q` để chạy nhiều game nhanh hơn.

**Chấm điểm:** Chúng tôi sẽ chạy agent của bạn trên layout `openClassic` 10 lần. Bạn nhận 0 điểm nếu agent timeout hoặc không thắng lần nào. Bạn nhận 1 điểm nếu thắng ít nhất 5 lần, hoặc 2 điểm nếu thắng cả 10 lần. Bạn nhận thêm 1 điểm nếu điểm trung bình lớn hơn 500, hoặc thêm 2 điểm nếu lớn hơn 1000. Bạn có thể thử agent của mình trong đúng điều kiện chấm bằng:
```bash
python autograder.py -q q1
```
Để chạy không đồ họa, dùng:
```bash
python autograder.py -q q1 --no-graphics
```
